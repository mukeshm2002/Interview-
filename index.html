2. What is a Java Full Stack Developer?
A Java Full Stack Developer works on both Frontend and Backend parts of a web application. Backend uses Java and Spring Boot to handle business logic and database operations, while Frontend uses HTML, CSS, and JavaScript for the UI and interactivity.

3. What are Classes and Objects?
A Class is a blueprint that defines properties and behaviors (methods). An Object is an instance of a class. Example: Car is a class, and myCar is an object with specific attributes.
4. Explain Method Overloading and Overriding
Method Overloading allows multiple methods with the same name but different parameters. Overriding occurs when a subclass provides a specific implementation of a method from its superclass.
5. What is Exception Handling?
Exception handling manages runtime errors using try-catch blocks. In my Bus Booking project, I created custom exceptions like NoRouteFoundException, BusNotFoundException, InsufficientSeatException, and SeatAlreadyBookedException for better user messages.
6. What is the difference between throw and throws?
‚Äòthrow‚Äô is used to manually throw an exception. ‚Äòthrows‚Äô declares exceptions that a method might throw, informing the calling method to handle them.
7. What are String, StringBuilder, and StringBuffer?
String is immutable, meaning changes create a new object. StringBuilder and StringBuffer are mutable. StringBuilder is faster (not thread-safe), while StringBuffer is thread-safe.
8. What are the main features from Java 8 to 25?
Key features include Lambda expressions, Stream API, Optional class, Modules, Records, Pattern Matching, Virtual Threads, and improved performance.
9. Explain Abstract Classes and Interfaces
Abstract classes can have both abstract and concrete methods. Interfaces only define method signatures. In Spring, interfaces like JpaRepository define contracts implemented by classes.
10. What is the Java Collections Framework?
It provides data structures like List, Set, Map, and Queue. Example: I used ArrayList to store employees, HashMap to map IDs to routes, and HashSet to avoid duplicate seats in my Bus Booking System.
11. What are Stream API and Lambda Expressions?
Lambda Expressions simplify code using short syntax. Stream API is used for processing collections (filter, map, reduce). I used Stream to filter available buses or employees in projects.
12. What is an Array? How did you use it?
An Array stores multiple elements of the same type. I used arrays for seat number tracking in my Bus Booking System before switching to dynamic lists.
13. What are some common pattern programs?
Example: Printing triangles, pyramids, and number patterns. It improves logic and nested loop understanding.

14. What is Spring Boot?
Spring Boot simplifies Java application development with auto-configuration, embedded servers, and less boilerplate code. I used it in all my backend projects for quick deployment and API creation.
15. Explain Spring Boot Modules
Spring Core (IoC), Spring MVC, Spring Data JPA, Spring Security, and Spring Test are key modules. Each supports a specific layer of the app.
16. Explain Spring MVC Architecture
Model-View-Controller: Model handles data, View displays UI, Controller manages requests. In my Employee System, Controller fetched employee data and displayed it using Thymeleaf templates.
17. What is Spring Security?
Spring Security provides authentication and authorization. I used it for login validation and role-based access in my Bus Ticket Reservation System.
18. How do you connect a database in Spring Boot?
By configuring application.properties with datasource details and using JPA repositories for CRUD operations. Example: Connecting to MySQL in Employee Management System.
19. What is RESTful API?
RESTful APIs allow communication between frontend and backend via HTTP methods like GET, POST, PUT, DELETE. I created REST APIs for all CRUD operations in my projects.
20. What are Design Patterns and how did you use them?
Design Patterns are reusable solutions to common problems. Example: Singleton for database connections, MVC pattern in project structure, and Factory pattern for object creation.
21. What is JPA and its interfaces?
JPA simplifies ORM with interfaces like EntityManager, JpaRepository, and CrudRepository. I used JpaRepository for easy database operations.
22. What is DispatcherServlet?
It is the front controller in Spring MVC that receives all requests and forwards them to appropriate controllers.
23. What is Tomcat Server?
Tomcat is a lightweight web server that runs Java web applications. Spring Boot uses an embedded Tomcat server to deploy applications easily.
